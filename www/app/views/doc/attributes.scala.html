API Builder supports the notion of 'attributes' that can be used to enhance code generation:

  • Attributes are globally unique and have URL friendly names.
  • Attributes have an optional description.
  • Attribute values can be set at the organization level, cascading to all of the applications that belong to that organization. Organization level values can be found by going to your organization, clicking '@lib.Util.OrgDetailsText' in the nav bar, then '@lib.Util.OrgAttributesText'.
  • A code generator can return a list of the attribute names on which it depends. API Builder will then send any matching attribute values when invoking the code generator.
  • View available attributes / create a new attribute
